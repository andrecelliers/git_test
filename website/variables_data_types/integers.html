IntegersÔÉÅ

In Python (as most programming languages), numbers are treated differently than strings. They are interpreted as numbers, therefore standard mathematical operations can be performed on them. Integers are whole numbers (as opposed to Floats). Any two integers can be declared and then used mathematically.

Note

Be careful of division in Python - the quotient of two ints could result in a float if the numbers cannot be divided evenly.

Declaring ints, as all other variables in Python, is simple syntax:

# Declaring basic integer values
rank = 10
eggs = 12
people = 3

Warning

When using an int in combination with Strings, it is important to cast the int as a string in order for it to print out correct. For example, print("My age is " + str(myAge)) casts the int myAge as a string within the context of the print statement.

The most common usage of ints is in for mathematical operations, such as:

length = 10
width = 5
height = 3
base = 10
side = 6
rectangleArea = length * width # the value of 30 would now be assigned to the variable area
triangleArea = (0.5 * base * height) # one half base multipled by height
squareArea = side**2 # exponents
print("The area of a the rectangle is " + str(rectangleArea))
print("The area of the triangle is " + str(triangleArea))
print("The area of the square is " + str(squareArea))

Which would result in:

The area of a the rectangle is 50
The area of the triangle is 15.0
The area of the square is 36
